FIFA World Cup odds via the-odds-api, served from Vercel.

Live (calls the-odds-api)

/api/worldcup — World Cup odds (TAB, h2h) /api/sports?soccer=true — soccer competitions

Stored (reads Turso, 0 credits)

/api/latest — latest stored featured snapshot /api/latest?list=true — snapshot history

Storing happens via /api/refresh (secret-protected; runs daily by cron). Trigger manually with /api/refresh?key=YOUR_CRON_SECRET, or a deep extra-markets sweep with /api/refresh?mode=deep&key=YOUR_CRON_SECRET.

For a single match, sweep all extra markets with /api/match?event_id=MATCH_ID&key=YOUR_CRON_SECRET (get the id from /api/worldcup or /api/latest).

Query options for /api/worldcup

  • regions — default au (Australian books incl. TAB)
  • markets — default h2h; e.g. h2h,totals,spreads
  • bookmaker — default tab; use all for every book
  • format — simple (default) or raw

Example: /api/worldcup?bookmaker=all&markets=h2h,totals

If you see an ODDS_API_KEY is not set error, add the environment variable in Vercel → Project → Settings → Environment Variables, then redeploy. Get a free key at the-odds-api.com.
